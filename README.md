# Pinnacle

Pinnacle theme for Awesome Window Manager.

## Requirements: 

 - Cantarell Font
 - Ranger
 - Picom (optional)
 - Feh (wallpaper)
 - Alacritty

## Setup

1. cp `settings.example.lua` to `settings.lua`. Edit accordingly. 

## Help

Press Super + H for help.
